## Usage
You can install this package from vs code and use inside vscode.

## Developers
```sh
$ cd myExtension
$ vsce package
# myExtension.vsix generated
$ vsce publish
# <publisherID>.myExtension published to VS Code Marketplace
# Or upload the file in market place
```
